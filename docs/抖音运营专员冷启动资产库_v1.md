# 抖音运营专员 · 冷启动资产库 v1

> 用途：本文件是"抖音运营专员"数字员工的核心知识资产，供选题引擎、剧本引擎、质检环直接调用。
> 分层原则：本库属于【结构层】（稳定，季度级更新）；热点话题、热门BGM、挑战赛、热词属于【风向层】，必须由 live 数据管线每日拉取（热点宝/巨量算数/搜索），两层在系统中分开存储，禁止混写。

---

## 0. 策略总纲（员工的"岗位认知"）

### 0.1 抖音与小红书的根本差异

| 维度 | 小红书 | 抖音 |
|---|---|---|
| 流量核心 | 点击逻辑：封面+标题决定生死 | 完播逻辑：前3秒+节奏决定生死 |
| 分发方式 | 双列信息流，用户主动选 | 单列推荐流，平台硬塞 |
| 北极星指标 | 点击率 → 收藏/互动 | 完播率 → 互动 → 转粉 |
| 内容寿命 | 长尾（搜索可持续引流数月） | 短促（首小时数据定流量池） |
| 质检重心 | 封面三米法则、标题钩子 | 前3秒hook、每15秒节奏点 |

**结论：抖音专员的一切产出，优先为"完播率"服务。** 标题不是点击器官，是完播引导和评论钩子。

### 0.2 制作三档定位（MVP 主打一、二档）

| 档位 | 形态 | 自动化程度 | 交付标准 |
|---|---|---|---|
| 一档 | 图文成片（图片轮播+TTS+字幕+BGM） | 全自动（FFmpeg 合成） | 成片直接发布 |
| 二档 | 口播片（TTS/数字人+b-roll素材+字幕） | 全自动~95% | 成片直接发布 |
| 三档 | 剧情/实拍类 | 半自动 | 剧本+分镜+AI素材包+剪映草稿，人工精剪10分钟 |

### 0.3 员工的日/周交付定义

- **每日交付**：3 个剧本（必须使用 3 种不同 Hook 类型）+ 1 条一档/二档成片 + 当日选题理由（100字内）
- **每周交付**：1 份《对标账号拆解周报》（结构见 §5.1）+ 1 份数据复盘建议（数据闭环上线后启用）

## 1. 黄金3秒 Hook 库（24式）

> 用法：剧本引擎每个选题强制生成 3 个不同类型的 Hook 候选 → 质检环按 §6 打分 → 取最高分进入剧本。每条含：开场台词模板 + 例句 + 适用赛道。

### A类 · 冲突/暴论开场（互动率最高，慎用频次）

**H01｜禁止式**
- 模板：千万别{行为}，尤其是{人群}
- 例句："千万别随便买空气炸锅，尤其是租房党"
- 适用：测评、避坑、泛知识

**H02｜暴论裁决**
- 模板：{常见事物}就是{颠覆性定性}
- 例句："90%的儿童保险都是白花钱"
- 适用：财经、教育、消费决策（注意：留证据支撑，否则掉口碑）

**H03｜否定纠错**
- 模板：你{日常行为}的方法，从第一步就是错的
- 例句："你煮鸡蛋的方法，从第一步就是错的"
- 适用：生活技巧、教程

**H04｜对立站队**
- 模板：还在{A}的人，真的该醒醒了
- 例句："还在用Excel拉报表的人，真的该醒醒了"
- 适用：效率工具、职场

### B类 · 结果/利益前置（转化型内容首选）

**H05｜成果先行**
- 模板：（直接展示成品/数据画面）+ 这是我{时间}做出来的
- 例句：画面先放装修完工实拍，"这是我花3万8装出来的60平"
- 适用：家居、手工、变现类

**H06｜利益承诺**
- 模板：这条视频帮你省/赚{具体数字}
- 例句："这条视频起码帮你省3000块冤枉钱"
- 适用：攻略、薅羊毛、消费指南

**H07｜倒计时挑战**
- 模板：{N}秒教会你{技能}
- 例句："30秒教会你手机拍出电影感"
- 适用：轻教程（视频总长必须兑现承诺）

**H08｜价格锚点**
- 模板：{低价}和{高价}的{品类}，区别到底在哪
- 例句："9块9和99的防晒衣，区别到底在哪"
- 适用：测评、电商种草

### C类 · 悬念/好奇缺口

**H09｜数字悬念**
- 模板：{N}个{方法/真相}，第{M}个90%的人不知道
- 例句："5个微信隐藏功能，第3个连客服都不一定知道"
- 适用：盘点、干货合集

**H10｜半截话**
- 模板：我后悔了，早知道{停顿/转画面}……
- 例句："我后悔了，早知道这个功能，去年就不用熬那么多夜"
- 适用：好物、工具、经验贴

**H11｜反差视觉**
- 模板：（画面冲击先行，台词后置）你敢信这是{反差事实}？
- 例句：先放成品惊艳画面，"你敢信这是AI十分钟做的？"
- 适用：AI、手工、改造类（一档图文成片的天然Hook）

**H12｜故事钩**
- 模板：上{时间}，我干了一件{定性}的事
- 例句："上个月，我干了一件全公司都说我疯了的事"
- 适用：剧情、IP人设号

### D类 · 身份点名/场景代入（精准流量）

**H13｜人群点名**
- 模板：做{职业/身份}的，把手停一下
- 例句："开实体店的老板，把手停一下，这条别划走"
- 适用：B端、垂类干货

**H14｜场景代入**
- 模板：如果你也总是{具体痛点场景}
- 例句："如果你也总是开会两小时、纪要写一晚"
- 适用：效率、职场、工具

**H15｜第二人称命令**
- 模板：先别{当前动作}，看完这条再说
- 例句："先别下单，看完这条再买不迟"
- 适用：种草拦截、比价

**H16｜阶段限定**
- 模板：第一次{做某事}的，这条能让你少走{N}年弯路
- 例句："第一次开抖音小店的，这条能让你少走两年弯路"
- 适用：新手教程

### E类 · 借势/共鸣

**H17｜热点嫁接**
- 模板：最近全网都在{热点}，但没人告诉你{你的角度}
- 例句："最近全网都在晒AI修图，但没人告诉你商用会侵权"
- 适用：一切赛道（热点从风向层取，本库只存句式）

**H18｜嘴替共鸣**
- 模板：终于有人把{大家不敢说的}说出来了
- 例句："终于有人把团建的真相说出来了"
- 适用：职场、情绪共鸣类

**H19｜提问开场**
- 模板：你有没有发现，{反常现象}？
- 例句："你有没有发现，超市的牛奶永远把最新鲜的放最里面？"
- 适用：泛知识、商业拆解

**H20｜自曝式**
- 模板：说出来丢人，我{反差自曝}
- 例句："说出来丢人，我做运营五年，上周才知道这个功能"
- 适用：拉近距离、干货铺垫

### F类 · 结构型（配合特定剧本）

**H21｜清单预告**
- 模板：{领域}的{N}个段位，看看你在第几层
- 例句："用AI的5个段位，大部分人卡在第2层"
- 适用：盘点、测试互动（评论钩极强）

**H22｜双向对比**
- 模板：同样是{事}，{A}和{B}的差距有多大
- 例句："同样是发朋友圈卖货，高手和普通人差距有多大"
- 适用：案例对比、教学

**H23｜限时稀缺**
- 模板：这个{方法/红利}，能用多久不知道，先码住
- 例句："这个免费额度能薅多久不知道，先码住"
- 适用：工具、政策、羊毛

**H24｜悬赏互动**
- 模板：评论区告诉我{问题}，下一条我拍{回报}
- 例句："评论区告诉我你最头疼的家务，下一条我用AI给你解决"
- 适用：系列内容、涨互动

## 2. 剧本结构库（按时长×类型）

> 用法：选题确定后，剧本引擎按内容类型选模板，逐时间块填充。所有口播语速按 4~5字/秒 折算台词字数。

### 2.1 模板A：15~20秒 口播干货（日更主力）

| 时间 | 模块 | 要求 |
|---|---|---|
| 0-3s | Hook | 从Hook库取，台词≤15字，字幕大字号 |
| 3-6s | 痛点确认 | 一句话让目标人群"对号入座" |
| 6-15s | 干货123 | 最多3个点，每点一句话+画面/字卡切换 |
| 15-20s | CTA | 互动指令（关注/评论钩/收藏理由）二选一，不贪多 |

### 2.2 模板B：30~45秒 口播进阶

| 时间 | 模块 | 要求 |
|---|---|---|
| 0-3s | Hook | 同上 |
| 3-8s | 建立可信 | "我是谁/我做到过什么"，一句话，不自吹超过8秒 |
| 8-30s | 三要点展开 | 每个要点 = 结论先行 + 一个例子，节奏点间隔≤8s |
| 30-38s | 反转/加码 | "但最关键的其实是……"（防滑走设计） |
| 38-45s | CTA | 评论钩优先（提一个有争议或开放的问题） |

### 2.3 模板C：图文成片（一档，全自动）

| 模块 | 规格 |
|---|---|
| 封面静帧 | 0-2s，大字报式标题（复用小红书封面工厂模板） |
| 图片轮播 | 8~12张，每张3~4s，每张配一句≤20字旁白+字幕 |
| 旁白节奏 | TTS，每张图一个独立信息点，禁止跨图长句 |
| 结尾帧 | 互动问题字卡+引导关注，停留3s |
| BGM | 从风向层热门BGM榜取，卡点切图（FFmpeg按节拍切） |

### 2.4 模板D：30秒 产品种草

| 时间 | 模块 | 要求 |
|---|---|---|
| 0-3s | 痛点场景 | 演/说一个具体使用困境 |
| 3-8s | 产品登场 | 一句话定位："直到我遇到了X" |
| 8-22s | 3个卖点演示 | 每个卖点=动作演示+一句结论，不念参数表 |
| 22-27s | 价格钩 | 锚点对比（"一杯奶茶钱"句式） |
| 27-30s | 行动指令 | 单一指令：点小黄车/评论扣1 |

### 2.5 模板E：60秒 剧情类（三档，半自动）

结构：冲突开场（0-5s，从H12/H18取）→ 冲突升级（5-25s，两次递进）→ 反转（25-40s）→ 价值点落地（40-52s，把产品/观点自然带出）→ CTA（52-60s）。
交付物 = 完整台词剧本 + 分镜表（§4）+ AI生成素材包 + 剪映草稿，明确标注"需人工精剪"。

## 3. 抖音标题/文案公式（16式）

> 抖音标题的三个职能：①完播引导 ②评论钩 ③搜索词埋点。每条标题必须至少命中一个职能，长度建议≤30字，关键词前置。

**T01 完播引导**："看到最后，{利益点}"｜"最后一个方法直接封神"
**T02 评论征集**："你遇到过吗？评论区聊聊"｜"评论区交出你的{X}"
**T03 争议站队**："不接受反驳"｜"敢说真话的来吵"
**T04 扣字互动**："想要{资源}的扣1"（注意：频繁使用有降权风险，每周≤2次）
**T05 搜索埋词**：把目标搜索词原样放进标题前8字，如"小红书开店流程 保姆级"
**T06 数字盘点**："{N}个{X}，建议反复观看"
**T07 身份点名**："{人群}必看"｜"打工人都给我看"
**T08 悬念补充**：标题补Hook没说完的信息，制造"必须看完"的信息差
**T09 反常识**："原来{颠覆认知的事实}"
**T10 时效紧迫**："{时间节点}前一定要知道"
**T11 自用背书**："自用{N}年，闭眼推荐"
**T12 教程承诺**："学会这招，{结果}"
**T13 系列连载**："{系列名}第{N}期｜{本期主题}"（养追更习惯）
**T14 提问式**："为什么{反常现象}？"
**T15 情绪共鸣**："谁懂啊，{扎心场景}"
**T16 资源清单**："{X}合集，码住慢慢用"

**话题标签策略**：3~5个为宜 = 1个泛领域大词 + 2个垂类精准词 + 1个风向层热点词（每日live拉取）+ 可选1个自建系列标签。

## 4. 分镜表输出规范

> 所有剧本（二档以上）必须附分镜表，字段固定如下，便于后续自动对接素材生成与剪映草稿。

| 字段 | 说明 |
|---|---|
| 镜号 | 01起编号 |
| 时间 | 起止秒 |
| 景别 | 特写/近景/中景/全景/字卡 |
| 画面描述 | 给视频生成模型/素材检索用的具体描述（含主体、动作、环境、光线） |
| 台词/旁白 | 精确到字（用于TTS） |
| 字幕 | 与台词一致或强调版 |
| 音效/BGM点 | 卡点、音效触发 |
| 素材来源 | AI生成（附生成提示词）/ 图库检索 / 用户实拍 / 截图录屏 |

**示例（模板A，15秒口播，节选）：**

| 镜号 | 时间 | 景别 | 画面描述 | 台词 | 素材来源 |
|---|---|---|---|---|---|
| 01 | 0-3s | 字卡 | 红底白字大字报"千万别这样发抖音" | 千万别这样发抖音，尤其是实体店老板 | 封面工厂渲染 |
| 02 | 3-6s | 录屏 | 手机界面：播放量两位数的后台截图，红框标注 | 你是不是也天天发，播放永远两位数 | 截图录屏 |
| 03 | 6-15s | 字卡×3 | 三张要点字卡依次切换 | 记住三件事：第一……第二……第三…… | 封面工厂渲染 |
| 04 | 15-18s | 字卡 | 结尾互动卡"你卡在哪一步？" | 你卡在哪一步，评论区告诉我 | 封面工厂渲染 |

## 5. 选题三板斧

**①对标复刻（主力，占60%）**：入职培训时收集 3~5 个对标账号 → 每周拆其近30天点赞前10的视频 → 按"Hook类型+结构模板+选题角度"三要素入库 → 换皮复刻（换行业案例、换人设口吻，结构保留）。
对标拆解周报字段：视频链接｜点赞/评论量级｜Hook归类（对应本库编号）｜结构归类｜可复刻角度｜不可复刻原因（如依赖出镜人设）。

**②热点借力（占20%）**：风向层每日拉取热点宝/巨量算数热词 → 质检环判断"热点与账号垂类的嫁接合理性"（强行蹭热点降权且掉粉）→ 用 H17 句式产出。

**③长青痛点（占20%）**：垂类高频搜索词（巨量算数搜索趋势，免费可查）→ 做成搜索型干货（T05埋词），吃长尾流量，对冲推荐流的不稳定。

## 6. 质检环硬规则（逐条可执行，违一打回）

1. 前3秒台词必须命中 Hook 库 24 式之一，且≤25字
2. 全程字幕，无一句裸音频
3. 每15秒至少1个节奏点（画面切换/反转/新信息块），由分镜表时间轴校验
4. 口播语速折算：台词总字数 ÷ 时长 ∈ [3.5, 5.5] 字/秒
5. 结尾必含且仅含1个互动指令（多指令=无指令）
6. 标题命中 §3 三职能至少之一，≤30字
7. 违禁过滤：广告法绝对化用语（最/第一/绝对/国家级）、医疗功效断言、"加微信/VX"等站外引导词，命中即打回
8. 一档成片附加：每张图停留 3~4s，旁白单图单句，BGM卡点误差≤0.3s
9. 时长档位匹配：干货≤45s，剧情≤60s，超时必须给出"为什么值得"的理由字段
10. Hook 多样性：当日 3 个剧本的 Hook 必须来自 3 个不同大类（A~F）

## 7. 发布与数据策略

**发布时间**（经验值，以账号实测为准）：工作日 6:00-8:00 / 11:30-13:00 / 18:00-23:00；周末上午整体后移1小时。新号固定时段发布，养算法预期。

**频率**：稳定日更1条 > 爆发式日更3条然后断更。一档成片的存在就是为了保证"断不了更"。

**冷启动逻辑**：发布后首小时进入初级流量池（约300-500播放），由完播率/点赞/评论/转发决定是否晋级下一池。员工应在发布后第2小时、第24小时各拉一次数据快照（数据闭环上线前由用户截图投喂）。

**及格线经验值**（因赛道浮动，仅作初始基准，由复盘闭环逐步校准为账号专属基准）：
- 15-20s视频完播率 ≥30%（优秀≥50%）；30-45s ≥20%
- 点赞率 ≥3%，评论率 ≥0.4%，转粉率 ≥0.1%

**DOU+测试逻辑**：只给自然流量已达标的内容买量（100元起测，定向相似达人粉丝），目的是测转化放大，绝不给烂内容续命——这条写进员工的"工作纪律"。

**复盘动作**（数据闭环）：连续3条完播率低于及格线 → 自动触发"Hook大类轮换+时长降档"策略调整，并在周报中说明调整逻辑。
